# Modern Image Gallery Pro

A responsive and interactive image gallery web application developed using HTML, CSS, and JavaScript. The project focuses on providing a modern user experience through a visually appealing interface, smooth animations, responsive design, and dynamic image management features.

## Features

* Responsive Masonry Gallery Layout
* Category-Based Image Filtering
* Real-Time Search Functionality
* Favorites Management
* Dark/Light Theme Toggle
* Image Download Support
* Fullscreen Image Viewing
* Smooth Animations and Transitions
* Mobile-Friendly Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Objective

The objective of this project is to create a modern and user-friendly image gallery that demonstrates front-end development concepts such as DOM manipulation, event handling, responsive design, local storage usage, and interactive user interfaces.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Nitish2507-cloud/Image-Gallery.git
   ```

2. Open the project folder.

3. Launch `index.html` using a local server such as VS Code Live Server.
