# Modern Image Gallery Pro

A modern responsive image gallery built using HTML, CSS, and JavaScript.

## Features

- Responsive Masonry Layout
- Category Filtering
- Search Functionality
- Light/Dark Theme
- Favorites System
- Image Download
- Fullscreen View
- Scroll Animations
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Folder Structure

ImageGalleryPro/
├── index.html
├── CSS/style.css
├── JS/script.js
└── images/
